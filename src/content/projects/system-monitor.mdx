This is a feature-rich and customizable system monitoring dashboard built with [egui](https://github.com/emilk/egui) in Rust.

The goal was just to have a functional system monitor to replace the GNOME system monitor because it's buggy on Ubuntu 24.04. Incidentally, this acquired many features while sill being lightweight (consumes slightly less than GNOME system monitor) while being much more informative, customizable and visually appealing (in my opinion).
